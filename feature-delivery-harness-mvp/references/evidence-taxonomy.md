# Evidence Taxonomy

```text
spec          | requirement, acceptance criterion, or scope statement
task          | task packet, work item, or handoff
test          | automated validation output
manual_check  | human usability or acceptance check
commit        | commit reference
ci            | CI check reference
review        | review reference
release       | release reference
dossier       | generated token economics dossier
other         | bounded synthetic evidence not covered above
```
